### 전략A-01. CK480 기법 ★최우선★
- **장·절**: 7장 15 (직장인을 위한 CK480)
- **대상 종목**: 당일 등락률 **+15% 이상** 상승한 급등주

  1. 1분봉 차트에서 **480분선(480개 1분봉 이동평균)** 까지 주가가 하락·접근

- **익절**: 진입가 대비 **+1~3%** (단기, 빠르게 대응)
- **손절**: **-2% 이내** (480분선 확실 이탈 시 즉시 손절, 원칙 엄수)
- **청산 금지 시간**: 오후 2:30 이후 적용 금지

- **변형**: 480분선 지지 → 240분선 돌파 시점이 더 강한 진입 신호. 480분선이 저항으로 역전 시 매도 전환.
- **데이터 요건**: minute_candles 480개 + 당일 daily_prices(등락률 계산). 전일 분봉까지 포함해야 480분 이평 계산 가능(당일+전일 연속 분봉 필요).

- **익절**: 명시 없음. 5·10분선 위 추세 유지 시 보유.
- **손절/청산**: **5·10분선 이탈** 시 즉시 청산
- **데이터 요건**: minute_candles 1분봉(또는 3분봉 집계)

### 전략A-13. 1분봉 240·480분선 지지/저항 필터

- **진입 조건**:
  1. 급락·조정 시 **240분선 → (더 깊으면) 480분선** 까지 눌렸다가
  2. 거래량 회복 + **반등 양봉** 시작 시 매수
- **매수 금지**: **240·480분선 이하**에서 상승 시도 시 이 선이 저항 → 480분선 상향 돌파(안착) 전까지 매수 금지
- **손절**: 240·480분선 재이탈 시 손절
- **데이터 요건**: minute_candles(당일+전일 분봉 연속 필요, 240·480분봉 계산)

### 전략A-14. 일봉 240·480일선 추세 필터 / 손절

  2. **480일선 이탈** = 추세 붕괴 시작 → 회복 전까지 **매수 금지**, 보유 중이면 **무조건 손절**
  3. 480일선 위에서만 단기 전략 진입 허용(상위 필터)

## 4. 책 전체 공통 원칙

### 청산 원칙: "이평선 이탈 시 매도"
책 전반에 걸쳐 익절·손절의 % 수치를 명시하는 경우가 드물다. 대신 일관된 청산 원칙이 반복된다:
- **단기(1분봉 기반)**: 5분선·10분선·20분선 이탈 시 즉시 청산
- **스윙(일봉 기반)**: 20일선 이탈 시 매도. 살짝 이탈 후 즉시 회복이면 보유 유지
- **추세 필터**: 240·480일선(일봉) / 240·480분선(1분봉) 이탈 = 추세 붕괴, 매수 금지 또는 손절

### 익절% 미명시 경향
- 대부분의 전략은 익절 목표가 "흐름 유지 시 보유 / 이평선 이탈 시 매도" 구조
- 명시 익절% 있는 전략은 **2곳뿐**:
  - **전략A-07 (20일선 눌림목)**: +10%
  - **전략A-01 (CK480)**: +1~3% (손절 -2% 동시 명시)

### 시간대 감각
- **오전 9~10시**: 테마/특징주 강약 판가름. 10시 이후 추격 금지.
- **9시 40분 부근**: 대형주 당일 고점 형성 빈도 높음 → 매도 타이밍
- **점심(12:00~14:30)**: CK480 기법의 유효 시간대
- **오후 14:30 이후**: CK480 적용 금지. 장 마감 변동성 증가
- **종가 무렵(14:30~15:30)**: 수급 기반 종가 베팅(기관·외국인 순매수 확인 필요 → C등급)

### 급등 정의 (정량화 필요)
책은 "급등주"라는 표현을 자주 쓰나 명확한 % 정의가 없음. 코드화 시 자체 임계 설정 필요:
- CK480: 당일 +15% (명시)
- 20일선 눌림목: 이전 급등(5~30일 내 +20~50% 수준으로 추정)
- 시초가 갭상승 위험 기준: +15% (명시)
- 5파 고점 매도: 아침 30~40분 내 빠른 상승

### 컴플라이언스 주의
자동매매 코드화 시 한국거래소 불공정 매매 9유형 경계:
- 분할 매수 10회 이상(❽), 취소정정 호가 과다(❾), 종가/시가 관여 과다(❺❻), 허수성 주문(❶)
- 자동화 코드에 주문 빈도·취소율·동시호가 관여 횟수 가드 필수
