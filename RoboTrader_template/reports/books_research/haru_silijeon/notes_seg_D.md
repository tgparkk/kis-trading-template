# 구간D 판독 (b3_097~b3_130)

강창권 『주식투자 단기 트레이딩의 정석』 — 마지막 구간.
이미지는 한 페이지를 좌(L)·우(R)로 분할 캡처한 형태이며, **우측(R)에 본문 오른쪽 끝이 잘려** 일부 문장이 중간에 끊김. 잘린 부분은 "(우측 잘림)"으로 표기. 숫자가 흐릿하면 "(불확실)".

판독된 챕터 구성:
- **5장 후반 ~ 프로그램 매매** (b3_097~b3_101)
- **6장 표지 + 직장인 단기매매** (b3_101~b3_102)
- **7장(15) 직장인을 위한 CK480 기법** (b3_102~b3_109)
- **7장(16) 직장인 스윙 매매 전략** (b3_110~b3_118)
- **부록: 실전 단기 매매 MTS 세팅법** (b3_119~b3_127)
- **b3_128~130: b3_127과 동일 페이지 반복 → 책 끝**

## 1. 프로그램 매수, 무조건 믿지 마라 (5장 후반, b3_097~b3_098)

- **요지**: 프로그램 매수는 초보에게 매수 신호처럼 느껴지지만, 그 자체가 상승의 보증 수표는 아니다. 단타 수급일 가능성도 있다.
- **사례**: 레인보우로보틱스 — 1분봉, 신고가 영역에서 프로그램 매수가 들어오며 장 마감 무렵 급락했다(2025.2.4 9:13 / 11:20 / 1분봉 사례). 프로그램 매수 후 짧은 시간 안에 매도로 돌아서면 주가가 급락.
- **판단 기준**: 프로그램 매도가 나오는지 **계속 체크**해야 한다. 수급 유입/이탈을 시간외·실시간으로 확인.
- **코드화**: 프로그램 매매 수급 데이터는 **외부데이터 의존**(KIS/증권사 프로그램 순매수 TR 필요) → 신뢰성·지연 이슈. 신호로 직접 쓰기보다 필터/체크용. 제약 큼.

## 2. 대형주 매매에서 프로그램 매수가 주는 신호 — 프로그램 매수와 골드존 (5장, b3_099~b3_101)

- **꼭지명**: "프로그램 매수와 골드존"
- **사례**: 포스코홀딩스(POSCO홀딩스) 1분봉, 2025년 7월 18일. 2차전지 섹터 강세, 장 시작과 동시에 프로그램 매수가 유입.
- **골드존 정의**: 분봉 상 하향 이탈한 뒤 늪지목을 형성한 구간에서 기회가 되고, 프로그램 매수가 유지되는 상황에서 조정을 거친 종목의 늪지 구간을 '골드존'이라 부른다.
- **진입 조건(추정)**:
  - 대형주에서 프로그램 매수 유입 + **20분선** 부근 지지
  - 신고가 영역에서 프로그램 매수가 짧은 시간 안에 자리잡고 **20분선** 위로 안착하는지 확인 (5~20분 사이 재진입 사례)
  - 포스코홀딩스 시가총액 10조 원 이상 같은 대형주 대상
- **사용 지표**: 1분봉, 20분선(20분 이동평균), 프로그램 순매수.
- **익절/손절**: 명시 수치 없음(1~3% 단기 수익 흐름 언급은 뒤 CK480에서 구체화).
- **코드화**: 20분선 지지 + 분봉 패턴은 코드화 가능하나, **프로그램 순매수 유입 판정**은 외부 수급데이터 의존 → 제약.

## 3. [7장-15] 직장인을 위한 CK480 기법 (b3_102~b3_109) ★핵심★

### 3-1. 점심값 벌기 단타 전략(CK480) — "점심시간 이후, 실전 단타 기회를 잡아라" (b3_103)

- **타깃 독자**: 직장인 — 시간과의 싸움, 점심시간(스캘핑 불가능) 외 시간 활용. 전업 트레이더가 아니어도 점심 무렵 짧게 대응.
- **이름 유래**: CK = 저자 이니셜(강창권), 480 = **480분 이동평균선(=장 시작 후 8시간선, 약 이틀치 1분봉 누적)**. 직장인처럼 제한된 시간에 쓰는 단타 전략.
- **종목 선정**: 당일 **15% 이상 상승**한 급등주(감속스러운 흐름) 중에서.
- **진입 조건**:
  - **480분선에 도달**한 후 지지선으로 작용, **10~20분 정도 짧게 머무는 자리**에서 매수
  - 오전 장 중에는 적용 어렵고, 점심시간(오후 12시 이후) 단기 반등 가능성이 높아질 때 유효
- **익절 / 손절 / 보유**:
  - 익절: **1~3%** 단기 수익 목표, 빠르게 대응 (오전이면 단타로 짧게)
  - 손절: **-2% 이내** (작은 손실)
  - 보유: 오후 12시 이후~2시30분 이전(불확실, 점심시간대 단타). "이후에는 적용하지 않는 것이 원칙"
- **CK480 매매 전략 핵심 포인트(본문 박스)**:
  - **종목 선정**: 당일 15% 이상 상승했던 감속스러운 급등주
  - **진입 조건**: 480분선에 도달한 후 지지받는 10~20분 정도 짧게 머무는 자리
  - **수익 목표**: 오전에는 단타, 오후 12시 이후 1~3% 짧은 수익 목표로 대응 가능
  - **주의사항**: 오후 2시30분 이후엔 적용 ✗. 480분선을 확실히 이탈하면 -2% 내 손절(원칙 엄수)
- **사례(차트)**: 에코프로비엠 1분봉(2025.8.12) — 주가 하락 후 480분선에서 5~30분 접근, 이후 240분선이 저항으로 작용.
- **사용 지표**: 1분봉, **480분 이동평균선**(핵심), 240분선(저항), 거래량.
- **코드화**: ★코드화 가능★ — 당일 등락률 15% 필터 + 1분봉 480분 MA(=480개 1분봉 이동평균) 지지 + 시간대 필터(점심) + 1~3% 익절 / -2% 손절. 외부데이터 의존 낮음(가격·거래량만). **단, "480분선" = 1분봉 480개선이라 당일+전일 분봉 데이터 필요**.

### 3-2. CK480 응용 — 240분·480분 이동평균선 (정배열 전환 후) (b3_107)

- **꼭지명**: "240분·480분 이동평균선 아래로 역배열을 만든 경우"
- **요지**: 240분·480분선 아래로 역배열이 된 종목은 단기 매수 타이밍이 급함(추격 위험). 장기 이동평균선 하단에서 주가가 빠지는 경우는 매물대. "첫 번째 반등은 미끼일 수 있다."
- **사례**: NHN벅스 1분봉(2021.12.17) — 240분선·480분선 아래에서 주가가 내려간 뒤 조정 본격화. 첫 반등 후 재차 하락.
- **진입 주의**: 역배열 구간 첫 반등은 추격 금지. 240분선 회복·정배열 전환 확인 후 진입.
- **코드화**: 240/480분선 정배열·역배열 판정은 코드화 가능. 가격 데이터만 의존.

### 3-3. CK480 — 점심시간 눌림목 대응 전략 (b3_108) / 480분선 저항 활용 (b3_109)

- **꼭지명**: "480분선에서 지지받은 후 240분선을 돌파하는 시점을 노려라" / "480분선에서 꺾인 흐름을 포착하라"
- **요지**:
  - 직장인은 장중 뉴스를 실시간 대응하기 어려워, 점심시간 **눌림목**을 공략.
  - **480분선 지지 → 240분선 돌파** 시점이 매수 포인트.
  - 반대로 **480분선이 저항/역할 반전**이 되면(저항선으로 작용) **매도/포지션 정리** 전환.
- **사례**: 한화오션 1분봉(2024.11.8) — 트럼프 당선/한국 조선업 수혜 테마, 480분선 지지 후 240분선 돌파, 1~3% 단기 수익 기회. SKC 1분봉(2025.1.12~13) — 480분선 저항 맞고 밀림 → 매도 포지션 전환.
- **코드화**: 480분선 지지→240분선 돌파(매수), 480분선 저항(매도) — 모두 가격·MA 기반 코드화 가능. 단 "테마/뉴스"는 사례 설명일 뿐 진입 트리거는 MA.

## 4. [7장-16] 직장인 스윙 매매 전략 (b3_110~b3_118)

### 4-1. 스윙 매매 기본 전략 — "급하지 않게, 기회를 기다리는 전략" (b3_111)

- **타깃**: 직장인 — 실시간 대응이 어려운 사람에게 유리한 타이밍. 매일 아침 눈 뜨면 급등 테마주 소식. 하지만 단기 급등은 추격 위험.
- **종목 선정**: 최근 강하게 시세를 분출했던 종목들 중 **3일 이상 조정**받고 **이동평균선에서 지지**를 형성하는 종목.
- **핵심 4 키워드**: **기간 조정 / 이동평균선 / 거래량 / 기준점 수렴**.
- **진입 패턴**: 상한가(또는 급등) 기록 후 조정 기간을 거쳐 **20일선에서 반등**하는 흐름. (꼭지: "상한가 기록 후 조정 기간을 거쳐 20일선에서 반등")
- **사례**: 다날 일봉(2025.3~7월) — 2025.6 스테이블 코인 테마 급등, 6.11 고점 후 6.24 20일선 부근 지지, 30% 조정 후 반등. (재용산업 일봉도 사례)
- **사용 지표/데이터**: **일봉**, 20일선·60일선·10일선(이동평균선), 거래량, 기간 조정.
- **코드화**: 급등(상한가/대량 상승) → 기간 조정(3일+) → 20일선 지지 반등 → 거래량 패턴. 일봉 기반 코드화 가능. 외부데이터 의존 낮음.

### 4-2. 스윙 — 상한가 후 60일선 반등 (b3_112~b3_113)

- **꼭지명**: "상한가 후 조정 기간을 거쳐 60일선에서 반등"
- **사례**: 제룡산업 일봉 — 2024.5.10 고점, 한 달 가까이(약 한 달) 조정 후 6.13까지 상승. **60일선 근처에서 지지** 확인 후 분할 매수. 60일선 지지 후 4일 동안 조정(불확실).
- **진입**: 급등 → 기간 조정 → 60일선 지지 확인 → 분할 매수. 거래량 줄어든 눌림목 주목.
- **코드화**: 60일선 지지 + 거래량 감소 패턴 → 코드화 가능(일봉).

### 4-3. 스윙 — 급등 후 거래량 감소하며 20일선 지지 (b3_113~b3_114)

- **꼭지명**: "급등 후 거래량이 감소하며 20일선 지지를 받을 때"
- **사례**: 노보노디스크 비만 치료제 '위고비' 국내 도입 테마, 관련주 나노엔텍/펩트론 등 — '데이트레이더' 매체 보도, HTS엔 반영 안 됨(미반영 정보 영향). 키움증권 HTS 시황장 뉴스 활용.
- **진입**: 급등 후 거래량 감소 + 20일선 지지 → 종가 무렵 확신 시 진입. 아침 HTS 미리 보고 진입 준비.
- **사례2**: 넥스틸 일봉(2024.7~11) — 2024.10.21 거래량 급증 급등, 10일선 이상 조정 후 240분/일봉 흐름, 약 12일간 기간 조정.
- **코드화**: 20일선 지지 + 거래량 감소 코드화 가능. **단 "뉴스/테마 미반영 정보"는 외부데이터(뉴스) 의존 → 제약**. 진입 트리거 자체는 가격·거래량.

### 4-4. 스윙 — 기관 수급에 올라타라 (b3_115~b3_118) ★수급 의존★

- **꼭지명**: "기관 수급에 올라타라 — 직장인 스윙 매매 전략" / "기관 수급을 동반한 240일선 돌파" / "기관 수급이 만드는 중장기 상승 추세" / "신규 종목도 기관이 매수하면 기회다"
- **요지**: 직장인은 실시간 이슈보다 **'흐름(수급)'**이 중요. **기관/외국인의 꾸준한 매수**가 눈에 띄게 유입되는 종목 주목. 개인 투자자가 앞다투어 매수하는 종목은 추격 위험.
- **진입 패턴**:
  - **기관 수급 지속 + 골든크로스 + 거래량** 동반 → 240일선 돌파 시 안정적 스윙 매수
  - 신규 상장 종목도 외국인/기관 1차 매수 타이밍 잡으면 기회 (외국인 순매수 유입 구간)
- **사례**:
  - 이수스페셜티케미컬 일봉(2024.10~2025) — 기관 수급 지속, 골든크로스, 240일선 돌파.
  - 한화비전 신규 상장(2024.2.6~), 외국인 1차 매수, 54,500원 신고가 부근.
  - 두산에너빌리티 일봉(2025.4~7) — SMR(Small Modular Reactor) 테마 주인공, 두 달 만에 +370% (74,100원). 외국인/기관 꾸준한 매수, 20일선/60일선 지지하며 상승. 10일·100일선 단위로 외국인·기관·대형 개인 매수.
- **사용 데이터**: **일봉, 기관/외국인 순매수 수급, 240일선·20일선·60일선·골든크로스, 거래량.**
- **익절/손절**: 명시 수치 부족. "20일선 한 번 이탈했다가 바로 회복" 시 추세 유지로 보유, "20일선 이탈+거래량 수반(주도주 변곡)" 시 매도 고려.
- **코드화**: MA·골든크로스·거래량은 코드화 가능. **기관/외국인 순매수 수급이 핵심 트리거 → 외부 수급데이터 의존(KIS/증권사 투자자별 매매동향 TR). 제약 큼** (저자 본인이 "직장인은 실시간 어렵다"며 수급을 일봉 단위로 사후 확인하는 방식 → 일봉 종가 기준 수급 데이터로 근사 가능).

## 5. [부록] 실전 단기 매매 MTS 세팅법 (b3_119~b3_127) — 키움증권 '영웅문 S#' 기준

전략이 아니라 **MTS(키움 영웅문 S#) 화면 세팅 가이드**. 코드화 대상이라기보다 저자 전략이 쓰는 **지표·데이터 정의**를 확인하는 용도.

### 5-1. MTS 이동평균선 (b3_120)
- HTS는 기본 120일까지지만 MTS는 추가 설정 가능. 단기 매매엔 **240분선·480분선** 추가가 핵심.
- 지표조건 설정창에 **5, 10, 20, 60, 120, 240, 480** 숫자를 넣어 저장.
- → CK480/응용 전략의 240·480분선 정의 근거.

### 5-2. MTS 기준선 (b3_121~b3_122)
- 단기 매매에서 활용하는 **기준선** = 전일 고가, 전일 저가, 당일 시가.
- **전일 고가** = 매물이 쌓인 저항선(이탈 시 손절매 쏟아짐), 기준선 이탈 시 매매 피하는 게 좋음.
- 설정: 종목차트 설정 → 도구 설정 → 기준선 → 전일 고가/전일 저가/당일 시가 라인 두껍게 ON 저장.
- → 진입/손절 기준에 "전일 고가/저가, 당일 시가" 활용.

### 5-3. MTS 실시간 조회 순위 (b3_122~b3_123)
- 단기 매매 종목 찾기: 영웅문S# 메뉴 → 국내 주식 → 관심종목 → 실시간 조회 순위. 검색량 급증 종목 = 시장 관심.
- 초보는 무작정 매수 ✗, **종목 선정 참고용**으로만. 눌림목 등 안전 타이밍 진입.

### 5-4. MTS 시간외 잔량 순위 (b3_123~b3_124, 화면번호 13003)
- 정규장 마감 후 시간외 단일가에서 매수/매도 잔량 순서로 표시. ETF+ETN 제외 설정 가능.
- 시간외 잔량 = 다음 날 시초가 흐름 예측. 호재성 재료 종목 잔량 주목.

### 5-5. MTS 시간외 등락률 순위 (b3_124, 화면번호 13004)
- 시간외 단일가에서 급등락 종목 등락률순. 정규장 종료 후 급등=익일 시초 갭상승 가능성. 외국인/기관 수급, 뉴스 연관성 고려해 진입.

### 5-6. MTS KRX/NXT 괴리율 순위 (b3_125~b3_126)
- 정규장 마감 후 **NXT(넥스트레이드, 대체거래소)** 마켓이 15:30~10:40(불확실, 오전 3시40분 표기) 추가 운영. KRX-NXT 가격 괴리율 순위.
- 괴리율 + 호재성 재료 + 거래량 동반 종목 주목. 단순 괴리율만으로 판단 ✗.
- **외부데이터 의존**: NXT 거래소 데이터 필요. KIS API NXT 지원 여부 확인 필요 → 코드화 제약.

### 5-7. 국내 주식 MTS 자동감시 주문 (b3_126~b3_127)
- 자동감시 주문 = 조건 미리 설정, 특정 가격/조건 도달 시 자동 매수/매도. 직장인이 장 못 볼 때 유용.
- 예: "삼성전자 65,000원 이하면 1,000주 매수" → 매수 조건 저장. 자동감시 ON 설정 후 조건 도달 시 자동 체결. 최대 3개월(불확실)까지 설정 가능.
- → CK480/스윙 전략의 지정가·조건부 진입을 MTS 자동주문으로 실행하는 운용 팁. **코드화 시 본 시스템의 지정가/조건주문 로직으로 대체 가능**.

### 5-8. 미국 주식 MTS 자동매수 주문 방법 (b3_127, 본문 마지막)
- 미국 주식도 매력적이지만 직장인은 실시간 대응 어려움 → 자동감시 주문 활용.
- 영웅문 S# → 주식 주문 → 미국 주식 자동감시 주문 → 신규 매수. 미국 시간 기준 밤 10:30 이후 매매를 자동화. 매수 조건+가격대+저장.
- → 본문 콘텐츠는 **여기서 종료**.

## 구간D 챕터 범위 요약

### 커버한 장/절
- **5장 후반**: 프로그램 매매 (프로그램 매수 믿지 마라 / 골드존) — b3_097~101
- **6장 표지** + 직장인 단기매매 도입 — b3_101~102
- **7장(소제목 15)**: 직장인을 위한 **CK480 기법** — b3_102~109
- **7장(소제목 16)**: 직장인 **스윙 매매 전략** — b3_110~118
- **부록**: 실전 단기 매매 **MTS 세팅법**(키움 영웅문 S#) — b3_119~127

### 코드화 가능 / 제약 요약
**코드화 가능(가격·MA·거래량 기반, 외부데이터 의존 낮음)**:
- **CK480 기법**(★최우선★): 당일 15%↑ 급등주 + 1분봉 480분선 지지(10~20분) + 점심 시간대 필터 + **익절 1~3% / 손절 -2%** + 오후 2:30 이후 적용 금지. 모든 트리거가 가격/거래량/시간. 240분선 저항·480분선 지지→240돌파 변형 포함.
- **스윙 매매**: 급등(상한가/대량) → 기간 조정 3일+ → 20일선/60일선/240일선 지지 반등 + 거래량 감소. 전부 일봉 MA·거래량. 코드화 가능.
- **기준선 룰**: 전일 고가/전일 저가/당일 시가 이탈 시 매매 회피 → 손절 기준 코드화 가능.

**제약(외부데이터 의존)**:
- **프로그램 매수/매도 수급** (골드존, 프로그램 신호) → 증권사 프로그램 순매수 TR. 지연·신뢰성 이슈.
- **기관/외국인 순매수 수급** (스윙 4-4 핵심) → 투자자별 매매동향 TR. 저자도 일봉 단위 사후확인 권장 → 종가 수급으로 근사 가능하나 핵심 트리거라 제약 큼.
- **뉴스/테마 미반영 정보** (위고비, SMR, 트럼프 테마) → 사례 설명용이며 실제 진입 트리거는 MA·거래량. 뉴스 자체는 코드화 제약(외부 뉴스 의존).
- **NXT(넥스트레이드) 괴리율 / 시간외 단일가** → 대체거래소·시간외 데이터. KIS API 지원 여부 확인 필요.
- **MTS 자동감시 주문** → 본 시스템의 조건주문/지정가 로직으로 대체 가능(외부 의존 아님, 운용 팁).

### 책 끝 도달 지점
- **본문 마지막 고유 페이지 = b3_127** ("미국 주식 MTS 자동매수 주문 방법"). 부록의 마지막 항목.
- **b3_128_L/R, b3_129_L/R, b3_130_L/R 은 b3_127_L/R 과 완전 동일한 중복 캡처** → 새 내용 없음. **책 본문은 b3_127에서 종료**됨(이후는 동일 페이지 반복).
- 참고: b3_100_L/R, b3_101_L 등에는 빈 페이지(장 표지 사이 여백) 존재. 6장 표지=b3_101, 7장(15) 표지=b3_102, 7장(16) 표지=b3_110, 부록 표지=b3_119.
