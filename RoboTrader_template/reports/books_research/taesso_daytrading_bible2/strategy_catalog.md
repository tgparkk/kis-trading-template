## 기법 A — 급등주 투매폭 매매법 (Ch1~2) ★시그니처
**개념**: 직전 고점을 강하게 돌파한 급등주가 고점 대비 "적정 투매폭(%)"만큼 눌릴 때 지지 확인 후 매수 → 저점대비 **+7% 반등** 노림. ~1,000개 표본·1.5년 추적·99% 신뢰구간 통계 기반(저자 시그니처 주장).

### 투매폭 매트릭스 (상승폭 → 고점대비 적정 하락폭 → +7% 반등 기대)
| 상승폭(저점→고점) | 적정 투매폭(고점대비 하락) | 평균 |
|---|---|---|
| 20~30% | 10~15% | 12.6% |
| 31~40% | 14~18% | 16.5% |
| 41~50% | 15~21% | 19.5% |
| 51~60% | 18~24% | 21.6% |
| 61~70% | 20~25% | 23.6% |
| 71~80% | 21~24% | 23.4% |
| 81~90% | 24~27% | 26.3% |
| 91~100% | 25~28% | 26.7% |
| 101%+ | 26~34% | 31.4% |
- 대략 **투매폭 ≈ 상승폭 × 0.25~0.30**. 진입가 = 고점 × (1 − 투매폭).
- **상한 게이트**: 고점대비 **30% 초과 하락 = 추세하락 의심**(큰 반등 기대난). 70%+ 상승주만 ~30%까지 허용.

### 종목선정
- 테마 대장주/시장 주도주/재료주, 직전 고점 강하게 돌파, 장대양봉 **거래대금 ≥ 200~300억**, 관리종목·우선주 제외.
- **MA20(20일선) 이탈 시 매매 금지/종료** (급등주 생명선).

### 진입 (지지확인 3중)
①RSI 과매도 탈출 순간의 저점 기준 / ②적정 투매폭 내 수평 지지선 지지 / ③거래량 ≤ 최대거래량 × **1/4** 급감.
→ 확인된 저점 **+1~3%**에서 분할매수. 선매수(미확인 매수) 금지.

### 청산
- **익절**: 저점대비 **+7%**부터 분할매도(짧게 +2~3%도). 대장주는 일부만.
- **손절**: 지지선(저점) 재이탈 시 전량.

### ★시간프레임 = 혼합(분봉 핵심)
종목선정·상승폭·투매폭 산정 = **일봉**. 진입·지지확인·청산 타이밍 = **15분봉**. → **분봉(15m) 백테스트 트랙 필요**. (메모리: 분봉 단타는 누적 전멸 영역 — 신중.)

## 기법 B — 종가배팅 (Ch3) ★일봉, 코드화 최적
**셋업 (일봉)**: 장대양봉(D0) → 1~2일 단봉조정(D1) → 익일 오전 상승파동.
- **D0 장대양봉**: 종가 **+7~25%**, 거래대금 **≥ 300억**, 위치=시세 초입(전고점/박스권/추세 돌파), 시총 **< 5,000억**, 관리/우선주 제외.
- **D1 단봉조정(진입일)**: 변동폭 **≤ 7~10%**, 전일비 거래량 **< 30%(급감, 이틀 연속 감소면 강)**, 캔들이 **[D0 장대양봉의 1/2가격 ~ D0 종가]** 사이에 위치(중간값 이탈·종가 초과 금지).
- **진입**: D1 **종가** 매수.
- **청산**: 익일 오전 **+2~3% 익절**(승률 30~50%, 손실폭 짧음). **손절 = 단봉 저점 이탈 시 전량**. (15분봉 슈팅거래량 1/2 하향돌파 시 분할매도 = 분봉 보조)
- ⚠️ 본문에 "손절 대신 불타기/물타기" 권유 있음 → **코드화 시 채택 비권장**(리스크).

### ★시간프레임 = 일봉 진입(종가)+익일 청산 → **기존 일봉 백테스트 프레임 직접 적용 가능**. 가장 유망.
