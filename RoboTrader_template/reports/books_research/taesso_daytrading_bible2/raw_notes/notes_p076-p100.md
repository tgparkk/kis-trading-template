## p076 (책 페이지 ~78) — "급등주 투매폭, 적정 투매폭 범위 지지 확인 방법" (소제목 시작)

- 소제목(녹색 헤더): **급등주 투매폭, 적정 투매폭 범위 지지 확인 방법**
- 직전 매매에서 매수 주가가 떨어졌을 때(투매) 어디까지 떨어지는지 = "적정 투매폭"을 확인하는 개념 도입.
- 주가가 더 떨어지지 않고 반등하며 기존 고가를 경신하면 "초기" 시점으로 봄 (지지 확인 → 매수 후보).

## p084 (책 페이지 86) ⭐ — **요약정리: 적정 투매폭 범위에서 주가 지지를 확인하는 방법**

- 본문 도입: **또 하나의 종목선정 요건은 차트상 직전 고점을 강력하게 돌파한 종목**이다. 직전 고점을 돌파한 차트는 기존 매수세력이 신규 매수세력에 손바뀜되는 **"초기"** 시점이며, 이 초기 시점에서 적정 투매폭 범위에서 반등도 잘 나오고 추가 급등도 많다.
- ⭐ **[요약정리] 적정 투매폭 범위에서 주가 지지를 확인하는 방법 (3가지):**
  1. **RSI 지표가 과매도 권에서 탈출하게 되었을 때, 과매도권의 저점을 기준으로 지지여부 확인** (주가가 현재 적정 투매폭 범위에 있을 경우)
  2. **적정 투매폭 범위 안에 특정 지지선이 존재할 경우, 그 지지선을 기준으로 주가 지지여부 확인** (주가가 현재 적정 투매폭 범위에 있을 경우)
  3. **최대거래량의 1/4 수준으로 줄어들 때 주가 지지여부 확인** (주가가 현재 적정 투매폭 범위에 있을 경우)
- "실제 예시사례를 보겠습니다." → 이후 페이지(대성파인텍 등) 사례 시작.

## p098 (책 페이지 100) ⭐ — **상승폭에 따른 적정 투매폭(하락폭) 범위 매핑 표 + 적정 투매폭 지지확인 방법(반복)**

- ⭐ **상승폭에 따른 적정 투매폭(직전 하락폭) 범위 — 코드화 핵심 테이블** (좌: 직전 상승폭 구간 → 우: 적정 투매 하락폭 범위, 단위 부분 추정·저해상도):
  - 직전 상승폭 **20~30%** 상승한 경우 → 적정 투매폭(하락폭) 그룹A **10~15%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **31~40%** 상승한 경우 → 적정 투매폭 그룹A **14~18%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **41~50%** 상승한 경우 → 적정 투매폭 그룹A **17~21%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **51~60%** 상승한 경우 → 적정 투매폭 그룹A **20~24%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **61~70%** 상승한 경우 → 적정 투매폭 그룹A **22~26%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **71~80%** 상승한 경우 → 적정 투매폭 그룹A **24~27%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **81~90%** 상승한 경우 → 적정 투매폭 그룹A **24~27%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **91~100%** 상승한 경우 → 적정 투매폭 그룹A **24~27%** (이후 저점대비 7% 이상 반등)
  - 직전 상승폭 **101% 이상** 상승한 경우 → 적정 투매폭 그룹A **24~27%** (이후 저점대비 7% 이상 반등)
  - ⚠️ **위 표의 하락폭 % 수치는 저해상도라 ±수% 오차 가능. 추세: 상승폭이 클수록 적정 투매(눌림) 하락폭도 커지고 약 20% 후반대에서 포화. "이후 저점대비 7% 이상 반등"은 전 구간 공통.** → 재확인 권장(FLAG: p098 수치).

  1. 보조지표 RSI가 과매도(과열)권에서 이탈하게 될 때, 해당 권의 저점을 기준으로 지지 여부 확인
  2. 일봉/분봉 상 적정 투매폭 범위 내에 지지선이 존재할 경우, 지지선 기준 지지 여부 확인
  3. 일봉/분봉 상 거래량이 최대거래량 대비 1/4 이하로 줄어들 때 매수해 보겠습니다

## p099 (책 페이지 103) ⭐⭐ — **요약정리: 급등주 투매폭 매매하는 방법 (코드화 최우선 규칙)**

- ⭐⭐ **[요약정리] 급등주 투매폭 매매하는 방법:**
  1. **직전 고점을 강하게 돌파하는 종목을 선정** (테마 대장주, 시장 주도주, "기대감" 있는 재료주)
  2. **고점확인 후 적정 투매폭 예상**
  3. **적정 투매폭 범위 내에 주가 지지 확인 후 매수, 손절선은 지지선 이탈 시**
     - **매수: 지지확인이 된 저점으로부터 +1~3% 부근에서 매수**
     - **손절: 지지확인이 된 저점을 재차 이탈할 때**
  4. **매도는 저점대비 7% 반등 시부터 분할매도**
  5. **한 차례 매매 이후 재매수 시 재료 및 대장주, 시장 주도주인지 체크**
  - 프로세스: **"고점확인 → 투매폭 예상 → 예상범위 근접 → 지지확인 → 매수"의 방식.**
- ⭐ **[Tip] 매수를 할 때 거래량이 급감한 상태에서 매수하는 것을 권장.**
  - **\* 주가가 20일선 이탈한 상태에서 매매금지.**

## 코드화 가능 규칙 요약 (엔지니어링 핵심)

- **종목선정(일봉 필터):** 단기간 큰 폭(20%~100%+) 급등 후 눌림 + **직전 고점을 강하게 돌파**한 테마 대장주/시장 주도주/재료주. **20일선 이탈 상태면 매매금지.**
- **적정 투매폭(눌림 하락폭) 예상(일봉):** 직전 고점 대비, 상승폭이 클수록 큰 하락폭 — 대략 20~30%상승→10~15%, 50%대→20~24%, 70%+→24~27% 눌림 (p098 표, 수치 재확인 필요).
- **지지 확인(15분봉, 3중 택1 이상):**
  1. **RSI 과매도권 탈출(이탈)** 순간, 그 과매도권 저점 기준 지지.
  2. 적정 투매폭 범위 내 **지지선** 기준 지지.
  3. **거래량이 최대거래량의 1/4 이하**로 감소(급감) 시.
- **매수:** 지지확인된 저점 **+1~3%** 부근. (거래량 급감 상태 권장.)
- **손절:** 지지확인된 저점(=손절선) **재차 이탈** 시 전량매도.
- **매도(익절):** **저점대비 +7% 반등 시점부터 분할매도.**
- **시간프레임:** 일봉으로 종목/추세/투매폭/20일선 필터 → **15분봉**으로 RSI·지지선·거래량 보고 진입/청산 = 분봉 단타.
