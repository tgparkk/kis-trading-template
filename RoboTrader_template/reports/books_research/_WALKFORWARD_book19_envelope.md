## 2. ★핵심 발견 — alpha의 regime 의존 (원래 우려와 정반대)

1. **엣지는 2~3 윈도우 집중 아티팩트가 아니다 (강건성 확인).** 8/11 양수, 그것도 alpha가 **2021H2~2024H2(약세·횡보·완만장)에 집중**(+12.9 / +4.6 / +35.7 / +38.0 / +17.1 / +11.0 / +10.1%). **원래 우려("full Sharpe 1.0은 2024~26 랠리 산물")를 정면 반박** — 랠리 *이전* 구간이 alpha 원천이다.

2. **★기존 OOS 홀드아웃이 오해를 낳았다.** "test 2024.7~2026.5 Sharpe 1.82 = 강한 OOS 엣지"는 사실상 **베타**(불장 편승)였지 alpha가 아니다 — 같은 구간에서 전략은 KOSPI를 **−25~−85% 하회**. 높은 test Sharpe는 메가 불장이 전부를 들어올린 것. (+워밍업 아티팩트로 test sub-window 측정 자체도 오염.)

3. **메가 불장에서는 드래그 (upside 캡).** 2025H2~2026H1 KOSPI +36/+97%일 때 전략은 +11/+11%만. **tp10% 캡이 승자를 +10%에 청산**하는 동안 지수는 +97% 질주 → 구조적 하회. = "확장 돌파 추격"이 강세장에선 평범, 비강세장에서 상대우위.

4. **깊은 약세 약점 재확인 (memory PARTIAL 판정과 정합).** 2022H1 Sharpe **−4.33 / −17.3%** — 지속 약세장에서 손실(유일 위안: KOSPI −22%보다 덜 잃음 alpha +4.6%). sl/tp 비대칭이 횡보 약세를 방어 못함.

5. **성격 = 횡보·완만약세 상대우위형, upside 캡.** 정확히 **RS 리더(나쁜 장 아웃퍼폼) 프로파일과 수렴**([[spike-2026-06-06-rs-leader-bad-market]]). MaxDD 양호(대부분 <16%, 2024H1만 25%).

## 3. 데이터 caveat

2025H2~2026 KOSPI 레벨이 극단적(→8476). 월별 궤적은 **부드럽고 단조**(3142→3486→4107→…→8476, 불연속 점프 없음)라 **데이터 내 일관된 실제 랠리**(아티팩트 아님). 단 절대수익 크기는 이 메가 불장 맥락에서 읽을 것. **윈도우 내 상대(alpha) 비교는 동일 DB·동일 유니버스라 유효.** 2021~2025초 KOSPI는 실제값과 정확 일치(검증됨).

## 4. 판정

**워크포워드 = 조건부 통과.** 엣지는 실재하고 랠리 이전부터 일관(강건성 ✅, 원래 fragility 우려 해소). **단 alpha는 regime 의존** — 횡보/완만약세 아웃퍼폼, 메가 불장 언더퍼폼, 깊은약세(2022H1) 손실. paper 관찰 전략(regime_gate=exclude_bear, 격리 1천만)으로는 적합 — 불장을 이기는 게 목적이 아니라 *나쁜 장에서 상대우위*가 thesis. **정식 leaderboard 등재는 여전히 페이퍼 실적 누적 후**(지시 유지). 메모리 "OOS 강건 신규 엣지 = envelope 하나"는 **"단, 그 엣지는 베타가 아니라 비강세장 alpha"로 정밀화**.
