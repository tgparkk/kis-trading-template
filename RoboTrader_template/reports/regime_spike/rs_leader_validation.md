## 3. 해석

**강건한 것 (파라미터 불문 성립):**
- SIDEWAYS 절대수익 5/5 config 양수(+5.07~+7.88%).
- OOS train·test 5/5 양수.
- bull·sideways·bear 전 국면 평균(+). → **"RS 리더가 횡보·평상장에서 절대수익을 낸다"는 핵심 주장은 강건.**

**강건하지 않은 것:**
- **약세장 에피소드 수익이 config마다 부호 반전**(2022·2024H2가 기준 config만 +, 나머지 −/≈0; 2026-03은 −4.9%~+2.9% 난반사). 표본 3개의 노이즈.
- 전 국면 **중앙값 음수·승률 38~42%(<50%)** → 소수 아웃라이어가 평균을 끌어올림(양의 skew, 변동성 큼).
- per-trade Sharpe 0.17~0.23 = 채택바(~0.6) 미달.
- 판정 자체가 **3 GO / 2 NO-GO** = 플래토 아닌 칼날.

## 4. 최종 결론 — 조건부(PARTIAL)

> **횡보장: 채택 후보 ✅ / 깊은 약세장: 미입증 ❌**

- 목표였던 "**나쁜 장에서도 오르는 종목**" 중 **횡보·완만한 약세**에 대해서는 RS 리더가 절대수익을 내며 *파라미터에 강건*하다 — 페이퍼 후보로 진전 가치 있음.
- 그러나 "**깊은 약세장에서도 수익**"은 **입증 실패** — 약세장 표본 3개에서 부호가 파라미터·구간 따라 뒤집히는 노이즈(소표본 함정). 구조적 진실과 일치: 깊은 폭락엔 상관계수→1로 디커플링 종목이 소멸(envelope 전략이 겪은 "약세장 후보 0").
- 엣지가 아웃라이어 의존·저Sharpe라 *그대로 라이브 부적합*.

## 5. 권고 (다음 단계, 별도 결정)

1. **범위 재설정**: "깊은 약세장 수익"은 목표에서 내리고, "**횡보장 RS 리더**"로 한정해 재설계(현실적·강건한 영역).
2. **청산 충실도 보강 후 재측정**: 현재 sl/mh 근사 — MA20 이탈 트레일링 청산(스펙 §4)을 커스텀 어댑터로 모델링하면 중앙값/승률 개선 가능(엣지 재평가 필요).
3. **샤프너 추가 실험**: "종목+ & 유니버스 breadth−"(디커플링) 필터를 얹어 약세 구간 선별 강화 → 약세 엣지 재검(단 표본 한계는 불변).
4. 채택 시에도 **격리 페이퍼 관찰 우선**(라이브 금지), 저Sharpe·아웃라이어 의존 명시.
