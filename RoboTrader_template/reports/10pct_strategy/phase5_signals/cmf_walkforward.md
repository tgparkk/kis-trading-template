# Phase 5 — CMF (Chaikin Money Flow) Walk-Forward + 레짐 검증

**판정: 폐기 — 1/5 PASS** (OBV와 정반대 결과, bull 레짐에서 역효과 확인)

## 요약

CMF는 동일한 5선 방법론을 OBV와 같은 daily_prices DB로 검증했으나 전면 실패. 모든 9개 파라미터에서 OOS Net이 음수이며 양의 OOS 윈도우 비율 0% (0/16). bull 레짐에서 오히려 강한 역효과(-0.57~-0.64pp/window), bear 레짐에서만 소폭 양수이나 거래비용(0.3%) 차감 후 전부 음수. 시총 × bull_high_vol 교차도 36셀 전부 음수. 1선(IS p-value 독립)만 통과.

## 데이터

- robotrader_quant.daily_prices 2,668,951 행 / 2,596 종목 / 2021-01-12 ~ 2026-05-22
- Winsorize: 1~99%
- 레짐: market_regime 테이블 미존재 → proxy로 derive (bull/bear × high/low_vol 4분류)
  - bull_low_vol 904,967행 / bear_low_vol 728,948행 / bull_high_vol 550,106행 / bear_high_vol 484,930행
- 시총 분위: Q1~Q4 cross-sectional (mc_q 유효 1,254,256행)
- CMF 라이브러리: `lib/signals/flow.py` `cmf()` 함수 (PIT 강제 확인)
  - MFM = (2×close - high - low) / (high - low), high==low → MFM=0 처리
  - MFV = MFM × volume, CMF_N = sum(MFV, N일) / sum(volume, N일)
  - rolling(window, min_periods=window) — 인과적 윈도우, 미래 참조 없음

## 원인 분석 (가설)

### CMF와 OBV의 결정적 차이
OBV는 종가 방향만 반영(단순 누적)하는 반면, CMF는 고/저가 대비 종가 위치(MFM)까지 반영. 한국 장내 특성상 장중 고점/저점 대비 종가가 낮게 마감(음봉 비율 높음)하는 경향이 있어 MFM이 구조적으로 음수 편향(-0.07 전후). 즉 CMF > 0인 종목은 장중 강세가 종가까지 유지된 "비정상적으로 강한" 종목 — 이는 **mean reversion** 대상이지 모멘텀 지속이 아닐 가능성.

### bull 레짐에서 역효과 패턴
bull 레짐에서 CMF > 0 종목 = 당일 급등 + 종가 고점 마감 → T+1 평균회귀 하락. OBV(누적 추세)와 달리 CMF는 단기 과매수 지표로 기능하는 것으로 추정.

### bear 레짐에서 gross 양수
bear 레짐에서 CMF > 0 종목 = 하락장 중 매수 압력 존재 → 상대적 강세 선별 가능성. 그러나 거래비용(0.3%) 차감 후 순수익이 미미하거나 음수로 실전 활용 불가.

## 권고

### Stage B — 불채택
CMF를 매수 필터 또는 매수 시그널로 활용하는 것은 현 데이터(daily_prices 2021~2026) 기준 권고하지 않음.

### 추가 탐구 가능 방향 (현 임무 외)
1. **역시그널**: CMF < -0.1 → 매도 회피 또는 숏 시그널 (bear에서 gross 양수 패턴 역이용)
2. **CMF 음수 → 매수 역발상**: CMF < threshold (역방향) 필터 — 평균회귀 전략 시 유효 가능성
3. **OBV와 결합 필터**: OBV 양호 + CMF > 0 (교집합) — CMF가 노이즈 제거 역할
