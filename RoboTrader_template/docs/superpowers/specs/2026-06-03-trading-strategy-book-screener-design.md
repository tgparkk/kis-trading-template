## 0. 사장님 원문 노트 (충실 보존 — 코드화의 기준선)

> 아래는 사장님이 전달한 노트 원문을 그대로 보존한 것이다. 코드화·검증은 이 노트를 기준으로 하며,
> 임의 추측·튜닝을 더하지 않는다(메모리 경고: "임계값 재튜닝 = 과적합").

### 0.1 매수후보(일봉) 개념
- 당일 거래량 급증 + 장대양봉 종목이 **눌림목**에 왔을 때 매수, 3% 내외 수익 목표.
- 당일 종목 선정 팁: 평소 거래량 대비 **최소 3배 이상 급증**(또는 급증 예상); 3/5분봉에서 상승폭 길고 하락폭 짧음.
- 장 초반 20% 이상 상승 종목은 웬만하면 제외. 일봉 100억 이상.
- 200일 신고가: 200일 이상 신고가 직전/달성부터 거래량 서서히 증가 + 등락폭 확대.
- Envelope(period=10, percent=10): 저항선 돌파 + 신고가 시 매매. 첫 신고가 또는 한두 달 조정 후 갱신 자리가 좋음. 박스권+엔벨 저항선 동시 돌파 첫 신고가 = 좋은 기회.
- 종가기준 200일 신고가 + 거래대금 최소 50억 이상. 한 달 내 100% 증가 종목 제외.

### 0.2 공식 조건식(HTS 조건검색식) — **코드화 기준**
```
A. 최고종가 : [일]0봉 전 종가가 200봉 중 최고종가
B. [일]0봉 전 Envelope(10,10) 종가가 Envelope 상한선 이상
C. 주가비교 : [일]0봉 전 시가 < 0봉 전 종가
D. 전일 동시간 대비 거래량 비율 100% 이상
E. 주가비교 : [일]0봉 전 종가 > 0봉 전 (고가+저가)/2
F. 5일 평균 거래대금(단위:백만) 5000 이상 (금일 제외)
G. 기간 내 등락률 : [일]0봉 전 1봉 이내에서 전일 종가대비 시가 7% 이상
H. 기간 내 등락률 : [일]1봉 전 1봉 이내에서 전일 종가대비 종가 10% 이상
I. 기간내 등락률 : [일]0봉 전 1봉 이내에서 시가대비 종가 3% 이상

조건식 : A and B and C and D and E and F and !G and !H and I
```
추가 진입 조건(산문): 종가가 이등분선[(고가+저가)/2] 지지 흐름, 일봉 대량거래량 조짐, 분봉에서 상승폭>하락폭 확인/예상 시 진입.

### 0.3 분봉(3/5분) 실행층 — **이번 범위 보류**
- 이등분선((당일고+당일저)/2): 강세=위, 약세=아래. 캔들이 이등분선 밑이면 안 됨("기준 몇 %?" 미정).
- 손절: 이등분선 지지 실패 / 상승폭<하락폭("길이 기준?" 미정). 접근 금지: 장중 +20% 급등, 상한가 다음날, 2마디 +20% 급등(단 이등분선 지지 시 시도).
- 가격박스(1분봉): 삼각이동평균(TMA, period=30)=중심선, 상/하한 = 중심 ± (조건부 편차평균 + 2·조건부std). 하한 지지 매수 / 중심 돌파 매수 / 상한 매도, 10분 무반응 손절.
- 볼린저밴드(3/5분, 20·2): 이등분선 위 + 밴드 밀집("밀집 기준?" 미정) + 상한 돌파 매수, 또는 첫 하한 지지 매수. 손절 −3% / 돌파양봉 저가·중심선 이탈, 익절 +3%.
- 눌림목 캔들: 주가↑·거래량↓ 추세 + 이등분선 위 조정 중 거래량 급감·캔들 축소 → 급감 거래량 상회 + 캔들 확대 시 매수. 기준거래량(당일 최다 거래량)의 1/4 수준 거래가 이상적, 1/2 이상이면 매수금지. 손절=이등분선/지지저점 이탈.
- → 이 층은 미정의 임계값이 다수이고(사장님이 직접 "?" 표기), 분봉단타는 18권 내내 전멸이라 보류. 임의 임계값으로 코드화 시 과적합 함정.

## 2. 진입 규칙 = 조건식 A~I (verbatim)
- 평가 시점 t = `df` 마지막 행(=0봉). 진입은 `entry_mechanism="market"` → 신호 다음봉(t+1) 시가. t+1 데이터 접근 금지(no-lookahead).
- 모든 지표는 trailing(과거~t). 최소 봉수 = `high_window + 2` (= 202; H가 `close[t-2]` 필요).

| 조건 | 코드 의미 | dataclass 필드(기본=책 원문) |
|---|---|---|
| A 200일 종가신고가 | `close[t] >= max(close[t-199..t])` | `high_window=200` |
| B Envelope 상단 돌파 | `close[t] >= SMA(close, env_period)[t] · (1+env_pct)` | `env_period=10`, `env_pct=0.10` |
| C 양봉 | `open[t] < close[t]` | — |
| D 거래량 비율 100%+ | `volume[t] >= volume[t-1] · vol_ratio` | `vol_ratio=1.0` (전일대비 일봉 프록시, 사장님 확정) |
| E 종가 > 이등분선 | `close[t] > (high[t]+low[t]) / 2` | — |
| F 5일 거래대금 ≥ 50억(금일 제외) | `mean(close·volume [t-5..t-1]) / 1e6 >= min_value_mil` | `value_window=5`, `min_value_mil=5000.0` |
| ¬G 갭상승 제외 | `not (open[t] >= close[t-1] · (1+gap_excl))` | `gap_excl=0.07` |
| ¬H 직전급등 제외 | `not (close[t-1] >= close[t-2] · (1+prior_surge_excl))` | `prior_surge_excl=0.10` |
| I 당일 시가대비 +3% | `close[t] >= open[t] · (1+intraday_gain)` | `intraday_gain=0.03` |

신호 = A∧B∧C∧D∧E∧F∧¬G∧¬H∧I.

- **거래대금(F) 단위 주의**: "단위:백만 5000" = 50억원. 데이터에 거래대금(amount) 컬럼이 있으면 그것을, 없으면 `close·volume`(원) → `/1e6`(백만) 환산해 5000과 비교. plan에서 daily 로더 컬럼 확인.
- **필드 노출 목적**: 기본값은 책 원문 verbatim. 멀티버스 그리드 스윕 대상으로 노출하되, 채택 판정은 **기본값(책 원문)** 기준. 임계값 튜닝 결과는 과적합 경고와 함께 참고만(메모리 1·2차 멀티버스 교훈).

## 3. 청산 = 드라이버 sl/tp/mh 스윕 (책 명목 앵커)
- 책 진짜 청산 = **장중 이등분선 트레일링 + 매수 대비 +3% 익절 / −3% 손절**. 장중 이등분선은 일봉 해상도로 표현 불가.
- → **진입만 충실, 청산은 근사**임을 report에 정직 명기(메모리 "분봉유도 청산을 일봉에 하드코딩 = 가짜 충실" 교훈 준수).
- exit-grid: `sl ∈ {0.03, 0.05}`, `tp ∈ {0.03, 0.05, 0.10}`, `mh ∈ {1, 2, 3, 5}` (책 명목 +3%/−3%를 그리드에 포함; 정확 값은 plan에서 확정).

## 4. 백테스트 (정본 멀티버스, daily)
- 드라이버: `scripts/book_portfolio_multiverse.py`
  ```
  --book trading_strategy_book --rule envelope_200d_high --granularity daily
  --universe top_volume:50 --K-list 3 5 10
  --max-per-stock 3000000 --initial-capital 10000000
  --exit-grid '{"sl":[0.03,0.05],"tp":[0.03,0.05,0.10],"mh":[1,2,3,5]}'
  --out D:\tmp\multiverse\book19_trading_strategy
  ```
- 다년 전구간(2021-01-01 ~ 2026 최신) 1회 + 3국면 달력창(BULL / BEAR(2022) / SIDE) 별도 `--start/--end` 실행.
  - caveat: 메모리상 통짜 달력라벨은 PIT `classify_daily`로 이행 중. 그러나 18권 비교가능성을 위해 동일 잣대(달력창) 사용하고, 한계를 report에 명기.
- 백그라운드 실행(daily는 분봉보다 빠르나 다구간·K스윕이므로 `run_in_background`).

## 5. 채택 판정 기준 (18권 잣대)
- Sharpe ≳ 0.6 (생존군 Elder 0.68 / Minervini 0.64), BEAR 비파국, 국면·유니버스 강건성.
- leaderboard 대비 위치 기록.
- **신호 희소성도 결론으로 인정**: 신호 n이 dino식(<20)이면 "국면 판정 불가·부적격"으로 정직 기록(억지 채택 금지).
- 예상: 조건식이 제한적(200일 신고가+엔벨+10%+3%양봉)이고 top_volume:50은 대형주풀이라 신호 희소 가능성 높음 — 그 자체가 유효한 발견.
