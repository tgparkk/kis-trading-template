## 0. 출처 노트 (spec 2026-06-03 §0.3 보존 — 코드화 기준선)

> 분봉(3/5분) 실행층. 노트 원문에 사장님이 직접 "?"로 표기한 미정 임계값이 다수.

- **이등분선**((당일고+당일저)/2): 강세=위, 약세=아래. 캔들이 이등분선 밑이면 안 됨("기준 몇 %?" 미정).
- **가격박스(1분봉)**: 삼각이동평균(TMA, period=30)=중심선, 상/하한 = 중심 ± (조건부 편차평균 + 2·조건부std). 하한 지지 매수 / 중심 돌파 매수 / 상한 매도, 10분 무반응 손절.
- **볼린저밴드(3/5분, 20·2)**: 이등분선 위 + 밴드 밀집("밀집 기준?" 미정) + 상한 돌파 매수, 또는 첫 하한 지지 매수. 손절 −3% / 돌파양봉 저가·중심선 이탈, 익절 +3%.
- **눌림목 캔들**: 주가↑·거래량↓ 추세 + 이등분선 위 조정 중 거래량 급감·캔들 축소 → 급감 거래량 상회 + 캔들 확대 시 매수. 기준거래량(당일 최다 거래량)의 1/4 수준이 이상적, 1/2 이상이면 매수금지. 손절=이등분선/지지저점 이탈.

- **청산 쪽 "?"는 장중 재량·이등분선 트레일링이라 분봉봉으로도 정확 표현 불가** → 임의 임계값 = 과적합.
  일봉에서도 같은 이유로 청산을 드라이버에 위임(18권 전체 관례). 따라서 청산은 멀티버스 `sl/tp/mh` 그리드로 근사(책 명목 +3%/−3% 포함).

### 1.2 검증 한계 (판정 기준에 사전 반영 — 정직)
- 분봉 데이터 `minute_candles` 커버리지 = **2025-02-24 ~ 2026-06-02 (≈15개월, 1,375종목)**. **전 구간 강세/횡보, BEAR 부재** → 약세장 강건성 측정 불가(일봉 A~I와 동일 한계).

## 2. 진입 규칙 (확정 임계값)

평가 시점 t = `df` 마지막 행(0봉). 진입은 드라이버 `entry_mechanism="market"` → t+1 시가. t+1 접근 금지(no-lookahead).
모든 지표 trailing(과거~t).

### 2.0 공통 — 이등분선 필터 (3전략 공유)
| 항목 | 확정값 | 비고 |
|---|---|---|
| 이등분선 | `(당일 누적고 + 당일 누적저) / 2`, 봉마다 갱신 | 노트 정의 |
| 이등분선 위 buffer | **0%** → `close[t] >= bisector[t]` | 노트 "밑이면 안 됨" |
| 진입 tolerance(지지/돌파) | **0.2%** | 분봉 호가 노이즈 흡수 최소값 |

- **★세션 인식 (분봉 룰 핵심 구현점)**: "당일 누적고/저", "당일 최다 거래량"은 **t와 같은 거래일(KST date)의 봉만** 사용해 계산. `df`에 `datetime` 컬럼이 있어야 하며, 룰 내부에서 `df[df.datetime.dt.date == t의 date]`로 당일 슬라이스. (일봉 룰엔 없던 요구사항.) `datetime` 부재 시 미트리거(방어).

### 2.1 전략 1 — `rule_price_box_tma` (1분봉, native)
| 항목 | 확정값 | dataclass 필드 |
|---|---|---|
| 중심선 | TMA(30) 삼각이동평균 | `tma_period=30` |
| 밴드 룩백 | 최근 **60봉** | `dev_window=60` |
| 밴드폭 | 중심 ± (`mean(\|close−TMA\|[t-59..t]) + 2·std(\|close−TMA\|[t-59..t])`) | `dev_k=2.0` |
| 진입 | `close ≤ 하한·(1+tol)` (하한 지지) **OR** `close[t-1] < TMA[t-1] & close[t] ≥ TMA[t]` (중심 상향돌파) | `tol=0.002` |
| 필터 | 이등분선 위(공통) | — |

- TMA(30) = SMA(SMA(close,15),15) 근사(삼각가중) 또는 표준 삼각이동평균. 구현 시 표준 정의 1개 채택해 docstring 명기.
- "조건부 편차평균/std" 해석: 노트에 "조건부" 의미 불명 → **전체 60봉 |close−TMA|** 의 mean+2·std로 단순화(가장 재현가능). report에 해석 명기.

### 2.2 전략 2 — `rule_bollinger_squeeze` (5분봉 리샘플)
| 항목 | 확정값 | dataclass 필드 |
|---|---|---|
| BB | (20, 2) on close | `bb_period=20`, `bb_k=2.0` |
| 밴드 밀집 | `bandwidth=(upper−lower)/mid ≤ median(bandwidth[최근 sqz_window봉])` | `sqz_window=100` |
| 진입 | 이등분선 위 + 밀집 + [`close ≥ upper`(돌파) **OR** `close ≤ lower·(1+tol)`(첫 하한 지지)] | `tol=0.002` |

- 5분 리샘플은 드라이버 `--minute-resample-freq 5min`. 룰은 리샘플된 df를 받는다.
- 밀집은 절대%(종목별 편차 큼) 대신 **상대(squeeze) 기준** = 최근 100봉 bandwidth 중앙값 이하.

### 2.3 전략 3 — `rule_pullback_volume_dry` (5분봉 리샘플) — **명시적 4단계(상승·하락·횡보·돌파)**
> 개정(2026-06-04): 초기 프록시 버전(`close[t]>close[t-6]` + 건조 + 확대)은 **되돌림(눌림) dip을 검출하지 않아** 단조 상승 후 확대봉에도 발사되는 충실성 갭이 있었음(사장님 지적). → 노트의 "눌림목"을 **상승→하락→횡보→돌파 4단계로 명시 검출**하도록 재설계. 사장님 확정 임계값.

평가 봉 t = **돌파(매수) 봉**. 직전봉 t-1 = 횡보(건조) 봉. 전부 당일 세션·PIT. `range = high−low`.

탐색: t 직전 `leg_window` 봉을 pre-breakout 윈도우로 보고, 그 안에서 국소 고점 `hi_idx`(=`P_high`)와 그 이후 되돌림 저점 `dip_idx`(=`P_low`)를 식별한다.

| 단계 | 확정 로직 | dataclass 필드 |
|---|---|---|
| **① 상승** | pre-breakout 윈도우 `[t-leg_window .. t-1]`의 국소 고점 `P_high`(=`hi_idx`). 그 직전 저점 대비 상승: `P_high ≥ min(low[t-leg_window..hi_idx]) · (1+rise_pct)` | `leg_window=12`, `rise_pct=0.02` |
| **② 하락(눌림)** | `hi_idx` 이후 되돌림 저점 `P_low = min(low[hi_idx+1..t-1])`(=`dip_idx`). `(P_high−P_low)/P_high ≥ dip_pct` **AND** `P_low ≥ bisector`(지지 유지). `hi_idx ≤ t-3`(되돌림 공간 확보) | `dip_pct=0.01` |
| **③ 횡보(건조)** | 되돌림 저점~t-1 구간 캔들 축소: `range[t-1] < mean(상승 leg range)`. 거래량 급감: `vol[t-1] ≤ day_max_vol · vol_dry_ratio` | `vol_dry_ratio=0.25` |
| **④ 돌파** | `close[t] > box_high`(횡보 박스 상단 = `max(high[dip_idx..t-1])`) **AND** `vol[t] > vol[t-1]` **AND** `vol[t] ≤ day_max_vol · vol_block_ratio` **AND** `range[t] > range[t-1]` **AND** `close[t] ≥ bisector` | `vol_block_ratio=0.50` |

- `day_max_vol` = t와 같은 거래일 봉들의 최대 volume(세션 인식, **현재봉 t 제외**=확장봉 자기참조 방지).
- 노트 "기준거래량의 1/4 이상적·1/2 이상 금지" → 1/4(`vol_dry_ratio`)·1/2(`vol_block_ratio`) 그대로.
- `need = leg_window + 3`. 4단계 어느 하나라도 불성립이면 미트리거.
- **핵심 차이**: ②하락(dip)을 명시 요구하므로 단조 상승 후 확대봉은 더 이상 발사되지 않음(실제 눌림이 있어야 신호).
- caveat: pre-breakout 윈도우가 세션 경계를 넘을 수 있음(전일 봉 혼입). day_max_vol만 세션 한정, 패턴 탐색은 trailing 윈도우 — 초기 프록시와 동일 한계, report 명기.

### 2.4 의도적 범위 제외 (충실성 위해)
- 청산 일체(10분 무반응·상승폭<하락폭·−3%/+3%·지지저점 이탈) → 드라이버 sl/tp/mh 근사(§1.1).
- "장중 +20% 급등 접근금지", "상한가 다음날", "2마디 +20%" → 장중 재량 게이트, 일봉 spec과 동일하게 보류. report 명기.
