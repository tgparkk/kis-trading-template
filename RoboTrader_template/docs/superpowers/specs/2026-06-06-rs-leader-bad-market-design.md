# RS 리더 — 나쁜 장에서도 오르는 종목 (설계 스펙)

- 날짜: 2026-06-06
- 상태: 설계 승인됨 (검토 대기) → 다음: 구현 플랜(writing-plans)
- 종류: 일봉 스윙 종목선정 전략 + 사전등록 검증 스파이크

## 0. 한 줄 요약

시장이 빠지거나 횡보해도 **자기 추세로 오르는 개별 종목**을 롱으로 잡는다.
시장 국면을 *예측*하지 않고, 종목의 **절대 상승 + 시장 대비 상대강도(RS)**가 곧 신호다.
먼저 **사전등록 검증 스파이크**로 엣지를 확인하고(엄격 GO 기준), 통과해야만 7번째 페이퍼 전략으로 승격한다.

## 1. 배경 / 의사결정 경로 (왜 이 설계인가)

이 설계는 일련의 의문을 데이터로 좁혀 도달한 결과다. 기록을 남겨 후속 세션이 같은 길을 다시 걷지 않도록 한다.

1. **목표**: 하락·횡보장에서도 *수익 나는* 전략. 기존 9전략(가상 6 + 라이브 3)은 전부 롱·강세추종이라 나쁜 장엔 "안 사고 비키는 방어"만 있고 능동 수익이 없음.
2. **인버스/숏 기각**: 한국 롱계좌라 인버스 ETF로만 표현 가능한데, 인버스 ETF는 변동성 decay(일일 리셋 복리침식)가 커 횡보장 보유 시 손실. "국면 예측"은 표본(약세장 3개)이 적어 통계적 증명 불가.
3. **데이트레이딩 기각**: 분봉(minute_candles) 커버리지가 **2025-02-24 ~ 2026-06-05 뿐**이고 그 구간 KOSPI는 +209%(MaxDD −19.9%)의 **인류사급 대세 상승장**. 진짜 약세장 0개·횡보장 거의 0 → "나쁜 장에서 통하는가"를 분봉으로는 **검증 불가**. 또한 사장님 이력상 분봉 단타는 전부 전멸.
4. **갭다운(약함 사기) 기각 — 데이터 근거**: "당일 시가 갭다운 → 종가 수익"을 일봉 41만 표본(2021~2026)으로 측정한 결과:
   - 갭다운(≤−1%)을 시장 방향별로 쪼개면: **KOSPI 하락일 평균 −0.73%(승률 39.4%)**, 보합일 +0.62%, 상승일 +0.77% (전부 비용 차감 전).
   - 즉 **나쁜 장에선 갭다운이 반등이 아니라 계속 하락** = 떨어지는 칼. 약함을 사는 컨셉(갭다운·투매폭·평균회귀)은 노리는 환경에서 구조적으로 손실.
   - 깊은 갭다운(≤−5%)은 시장 무관 −0.39% (악재 연속). taesso 『투매폭』 부적격 결론과 정합.
5. **결론(데이터가 가리킨 방향)**: 나쁜 장에서 살아남는 건 "약함을 사기"가 아니라 **"강함을 사기"** — 시장이 빠져도 오르는 RS 리더를 롱으로. 이게 본 설계.

## 2. 컨셉

- **방향 중립 추세추종을 종목 단위로**: 시장 방향이 무엇이든, *그 종목*이 절대 상승 추세이고 시장을 아웃퍼폼하면 후보.
- **국면 예측 불필요**: 종목 신호가 곧 판정. 시장 게이트 없음.
- **깊은 폭락 = 자연 방어**: 상관계수→1이면 조건 충족 종목이 0으로 수렴 → 거래가 저절로 줄어든다(능동 방어 아닌 *부재*, envelope 전략이 겪은 그것). 이는 버그가 아니라 정상 동작.

## 3. 선정 룰 (EOD 스크리너, 일봉)

기존 `RuleScreenerBase` + 전략별 어댑터 패턴 재사용 (per-strategy EOD 스크리너 구조, 커밋 d70e188 계열).

종목별 일봉(조정종가)으로 다음을 모두 충족 시 후보:

1. **유동성**: 일평균 거래대금 ≥ 10억 · 가격 1,000 ~ 500,000원 (기존 1차필터 재사용)
2. **절대 상승추세** (실제로 오르는 중):
   - 종가 > MA(60)
   - MA(20) > MA(60)
   - 60일 수익률 > 0
3. **상대강도 — 횡단면(cross-sectional, 지수 불요)**:
   - 종목 120일 수익률의 **유니버스 전체 대비 백분위 랭크**가 상위 분위 (예: 상위 20~30%) — O'Neil RS 레이팅 방식.
   - (선택 보강) 종목 120일 수익률 > 유니버스 중앙값 (= 시장 아웃퍼폼의 지수-불요 근사).
   - ※ KOSPI 지수 대비 RS(`mkt_rs` 필터)는 *부가 변형*으로 교차검증(실제 KOSPI 2021~ 가용).
4. **나쁜장 샤프너 (옵션 — 검증서 on/off 비교, 지수 불요)**:
   - 최근 20일: 종목 수익률 > 0 **이면서** 유니버스 20일 중앙값(=breadth) < 0 (시장 약한데 이 종목은 상승 = 디커플링).
5. RS 랭크 상위 **K개** 선정.

> 파라미터는 기본값으로 시작하되, 검증 단계에서 **소수 그리드**만 스윕(과적합 방지): MA 기간(60/120), RS 윈도우(120/60), RS 분위(상위 20~30%), K(5/10).

## 4. 진입 / 청산 / 사이징

- **진입**: 신호 다음 거래일 시가 (일봉 스윙, no-lookahead).
- **청산** (먼저 닿는 것):
  - 종가 < MA(20) — 추세 이탈
  - −8% 손절
  - RS 음전환 (종목−KOSPI 수익률 차 < 0)
  - max_hold = 스윙 (기본 20~40거래일, 검증서 확정)
- **사이징** (기존 6 페이퍼 전략과 동일 골격):
  - 페이퍼 독립자본 1천만 (`VIRTUAL_CAPITAL_PER_STRATEGY`)
  - 등비중 K (기본 5~10)
  - 종목당 최대 300만 (`max_per_stock_amount`)
  - `max_daily_trades` 5
- `holding_period = "swing"` (EOD 일괄청산 스킵)
- regime_gate = none (나쁜 장에도 끄지 않는 게 컨셉). RS는 §3 횡단면 정의 사용(지수 불요). 라이브 시장방향 급락가드(`regime_index`)는 KOSPI 실시간으로 두되 RS 판정과는 무관.

## 5. 데이터

- 일봉: **robotrader_quant.daily_prices** (조정종가 — adj_factor 곱하지 않음; SSOT 정책)
- **시장지수/국면 = 실제 KOSPI** (검증 완료): robotrader.daily_prices `stock_code='KOSPI'` 가 **2021-01-04 ~ 2026-05-29(1324행)** 보유. 기존 `_build_daily_regime_map`(KOSPI + breadth → classify_daily)·`mkt_rs` 필터가 그대로 사용 가능 → 별도 합성지수 불필요(직전 우회안 폐기).
- 횡단면 RS(`rs_rank`)는 애초에 지수가 필요 없음(유니버스 내 백분위). `mkt_rs`(KOSPI 대비 아웃퍼폼)는 실제 KOSPI로 보강 가능.
- 검증 기간: **2021 ~ 2026** (진짜 약세장 포함: 2022 깊은약세, 2024H2 충격형, 2026-03 V급락). KOSPI·일봉 모두 전 구간 커버 확인됨.
- 국면 라벨: 기존 `core/regime/regime_classifier.py` `classify_daily` (PIT, 파라미터 기본값 고정 — 이번엔 라벨 가치 측정이 아니라 *전략의 국면별 성과 분해*에만 사용). `book_portfolio_multiverse._build_daily_regime_map(start,end)`를 그대로 호출(실제 KOSPI + breadth 패널). 추가 구현 없음.

## 6. 검증 루브릭 (사전등록 · 엄격 — 결과 보기 전 확정)

이 전략은 cross-sectional 모멘텀이라 **momentum crash(음의 skew)** 리스크가 있다. 그에 맞춘 자를 쓴다.

### 측정
- **주지표 — 국면별 절대수익 분해**: `classify_daily`로 bull/sideways/bear 구간을 나눠, 각 구간의 **절대수익(비용 차감 후)**·승률·거래수. 목표의 직접 측정.
- **벤치마크 우위**: 나쁜 장(sideways·bear) 구간에서 KOSPI 및 기존 전략 대비 우위.
- **momentum-crash 스트레스**: 2022 깊은약세 + 2026-03 V급락에서 MaxDD·worst stretch.
- **OOS**: leave-one-bear-episode-out (2022 / 2024H2 / 2026-03 등 개별 제외 검증) + train(2021~2024.6)/test(2024.7~2026.5) 분할.
- **과적합 보정**: 파라미터 소수 그리드만, **plateau**(고원) 확인 + Deflated Sharpe Ratio.
- **비용**: 왕복 ~0.2~0.3%(세금·수수료·슬리피지) 반영.

### GO 기준 (전부 충족 시에만 승격)
1. **SIDEWAYS 구간 절대수익 > 0** (비용 차감 후), **≥2개 OOS 분할**에서 유지.
2. **BEAR 구간 비파국** — 절대수익이 깊은 음수가 아니거나, 후보가 비워져 방어적으로 동작.
3. **나쁜 장에서 KOSPI·기존 전략 대비 우위**.
4. **leave-one-bear-episode-out 생존** (한 약세장 빼고도 결론 유지).

> NO-GO면 전략을 만들지 않는다. "거짓 GO보다 거짓 NO가 싸다"는 원칙. 사장님 이력의 in-sample 착시·단일표본 과적합 함정을 사전등록으로 차단.

## 8. 정직한 한계 (설계에 내장)

- **깊은 폭락엔 거래 거의 없음**: 정상 동작이지 결함 아님. "나쁜 장 수익"은 주로 *횡보·완만한 약세*에서 나오고, 깊은 패닉(상관계수→1)에선 방어(미거래)가 최선.
- **momentum crash 노출**: 약세 바닥 V반등 시 직전 강세주 급락 → 빠른 청산(MA20 이탈·−8%·RS 음전환)로 방어하되 완전 제거 불가.
- **약세장 표본 소수**: 2021~2026 약세장이 몇 개뿐이라 BEAR 결론은 "증명"이 아니라 "방향성 증거". OOS·leave-one-out로 최대한 방어하되 한계를 인정.
- **기존 전략과 일부 중복**: minervini(RS-ish)·momentum 계열과 겹칠 수 있음. 차별점 = 명시적 RS-vs-시장 + 무게이트 + *나쁜 장 절대수익으로 검증*. 검증서 기존 전략 대비 증분 가치 확인 필요(GO 기준 3).

## 9. 비범위 (YAGNI)

- 분봉/장중 실행 (데이터상 검증 불가).
- 인버스/숏.
- RS 파라미터 대규모 튜닝.
- 라이브(실거래) 배선 — GO 후 페이퍼 관찰 우선.
