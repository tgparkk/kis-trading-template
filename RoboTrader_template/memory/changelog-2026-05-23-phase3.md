## 핵심 패턴 발견

### 1. 진입 시간 이동(09:30→10:00)은 부분 효과 있음
- 최저점 중앙값: -3.3% → -1.99% (1.31%p 개선)
- 손절 -0.8% 도달률: 85.6% → 79% (6.6%p 감소)
- 그러나 여전히 79%가 -0.8% 손절선에 도달 → 근본 문제 해결 미완

### 2. Pullback 후 반등(방식 B)은 역효과
- 하락 후 첫 양봉 진입이므로 이미 1%+ 하락한 상태
- 이후 추가 하락 가능성 여전히 높음 (-0.8% 도달률 82%)
- 30분 후 평균수익 -0.45% (방식 A -0.18%보다 나쁨)
- 매매 수도 32% 감소 (기회비용 증가)

### 3. 승률 구조 문제 — 매도 조건만으로 해결 불가
- 방식 A 최상위 셀도 승률 47.5~50% → 통과 기준 55% 미달
- 손절 넓혀도(2.0%) 승률 50% 수준 = 비용(0.41%) 차감 후 음수
- Phase 2에서 09:30 진입 → Phase 3에서 10:00 이동으로도 해결 안 됨
- **결론: 종목 선정 신호 자체의 방향성 정확도 부족**

### 4. 종목 선정 기준 재검토 방향
- `ret_20d_pct >= 25 AND atr_20d_pct >= 8` = 과거 20일 강세 + 변동성 높음
- 4~5월 KOSPI 약세장에서 이 조건이 평균회귀(하락) 종목을 선별하는 역기능
- 추세 방향 필터 추가 필요: 예) 당일 갭업 확인, 시장 강도 필터, 섹터 모멘텀
