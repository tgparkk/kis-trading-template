### 1. Opening Range Breakout (ORB)

#### 종목 선정 (장 시작 전 스크리닝)
| 조건 | 기준 | 구현 |
|------|------|------|
| 전일 거래대금 | 상위 100위 이내 (유동성 확보) | `avg_volume_20d * avg_price > threshold` |
| 프리마켓 갭 | ±1~4% 범위 (너무 크면 제외) | `abs(premarket_price / prev_close - 1)` |
| ATR(14) | 전일 종가의 1.5% 이상 (변동성 충분) | `atr_14 / prev_close > 0.015` |
| 뉴스/이벤트 | 실적발표·FDA승인 등 카탈리스트 보유 종목 우선 | 뉴스 API 연동 |
| **한국 추가** | KOSDAQ 중소형주 (변동성↑), VI 미발동 종목 | 시가총액 1,000억~2조 |

#### 매수 타이밍
```
[15분 ORB 기준]
1. 09:00~09:15 (한국) / 09:30~09:45 (미국) 고가·저가 확정
2. 롱 진입 조건 (모두 충족):
   - 현재가 > ORB 고가 (종가 기준 돌파, 또는 1틱 돌파)
   - 현재 거래량 > 20일 평균 동시간대 거래량 × 1.5
   - 현재가 > VWAP
   - 5분봉 20EMA 기울기 > 0
3. 숏 진입 조건:
   - 현재가 < ORB 저가
   - 거래량 조건 동일
   - 현재가 < VWAP
```

#### 매도 타이밍
```
[익절]
- 1차 목표: ORB 레인지 높이 × 1.0 (진입가 기준 측정이동)
  → 50% 물량 청산
- 2차 목표: ORB 레인지 높이 × 2.0 또는 전일 고가/저가
  → 잔여 물량 청산

[손절]
- ORB 반대편 (롱이면 ORB 저가 - ATR×0.3)
- 또는 진입가 대비 ATR(14) × 1.0

[시간정지]
- 14:30 (한국) / 15:30 (미국) 이후 무조건 청산
- 진입 후 30분 내 목표가 미도달 시 청산 검토
```

### 2. VWAP 평균회귀 전략

#### 종목 선정
| 조건 | 기준 | 이유 |
|------|------|------|
| 일평균 거래대금 | 50억원+ (한국) / $10M+ (미국) | VWAP 신뢰도 |
| 당일 추세 확인 | 시가 > VWAP (롱) 또는 시가 < VWAP (숏) | 편향 방향 설정 |
| 변동성 | ATR(14)/종가 > 1.5% | 충분한 이탈-회귀 폭 |
| 기관 거래 비중 | 높을수록 VWAP 앵커링 강화 | 한국: 외국인+기관 거래비중 30%+ |

#### 매수 타이밍
```
[VWAP 리버전 롱]
1. 당일 편향 확인: 시가 > VWAP → 롱 편향
2. 진입 조건 (모두 충족):
   - 가격이 VWAP까지 풀백 (VWAP ± 0.1% 이내 터치)
   - RSI(2) 5분봉 < 30 (과매도)
   - 직전 캔들이 VWAP에서 반등 양봉 (확인 캔들)
   - 거래량이 풀백 구간에서 감소 후 반등 시 증가

[VWAP First Kiss]
1. 시가부터 VWAP 크게 상회한 채 움직임
2. 장중 첫 번째 VWAP 리테스트에서 진입
3. 두 번째 이후 리테스트는 효과 감소 → 무시
```

#### 매도 타이밍
```
[익절]
- VWAP 상방 1 표준편차 밴드 도달 시 50% 청산
- VWAP 상방 2 표준편차 밴드 도달 시 전량 청산
- 또는 당일 고가 근처 (±0.3%)

[손절]
- VWAP 반대쪽 0.3~0.5% 이탈 시
- 금액 기준: $0.10~$0.20 (미국 유동주), 호가 2~3틱 (한국)

[시간정지]
- 장 마감 30분 전 무조건 청산
```

### 3. 갭 트레이딩 전략

#### 3-1. 갭필(Gap Fill) 전략

#### 종목 선정
| 조건 | 기준 | 비고 |
|------|------|------|
| 갭 크기 | -0.15% ~ -0.6% (소~중규모 갭다운) | 대형 갭(>1%)은 필 확률 낮음 |
| IBS | 전일 IBS < 0.25 | `(close - low) / (high - low)` |
| 대상 | SPY/QQQ 또는 KOSPI200 ETF | 지수 ETF가 가장 안정적 |
| 전일 패턴 | 전일 음봉 (추가 과매도 확인) | 선택적 필터 |

#### 매수 타이밍
```
1. 장 시작 전 갭다운 크기 확인
2. 시가에 즉시 매수 (갭 -0.15%~-0.6% 범위)
3. 또는 첫 5분 캔들 종가에서 양봉 확인 후 매수
```

#### 매도 타이밍
```
[익절]
- 전일 종가 회복 시 (갭 100% 필) → 전량 청산
- 또는 갭의 75% 회복 시 → 보수적 청산

[손절]
- 시가 대비 -0.5% 추가 하락
- ATR(14) × 1.0

[시간정지]
- 11:30 (한국) / 12:00 (미국)까지 갭필 미완료 시 청산
```

#### 3-2. 갭앤고(Gap and Go) 전략

#### 종목 선정
| 조건 | 기준 |
|------|------|
| 갭 크기 | +4% 이상 갭업 |
| 거래량 | 상대거래량(RVOL) > 2.0× |
| 카탈리스트 | 실적·뉴스·FDA 등 명확한 이유 |
| 시가총액 | $500M~$10B (미국) / 5,000억~5조 (한국) |
| 유동주식비율 | 낮을수록 급등 가능성↑ (Float < 20M 선호) |

#### 매수 타이밍
```
1. 시가 확인 후 첫 5분 고가 확정
2. 5분 고가 돌파 + 거래량 동반 시 진입
3. 또는 첫 풀백 후 불플래그 패턴 상단 돌파 시 진입
   - 풀백 깊이: 갭업 폭의 30~50% 이내
   - 풀백 시 거래량 감소 필수
```

#### 매도 타이밍
```
[익절]
- 프리마켓 고가 도달 시 50% 청산
- 나머지는 5분봉 트레일링 스톱 (직전 5분 저가)

[손절]
- VWAP 하향 이탈 시
- 또는 풀백 저점 하회 시
```

### 4. RSI(2) 평균회귀 (당일~단기)

#### 종목 선정
| 조건 | 기준 | 비고 |
|------|------|------|
| 추세 | 종가 > 200일 SMA | 상승추세 종목만 |
| 시가총액 | 대형주 우선 (S&P 500 구성종목, KOSPI 200) | 평균회귀 신뢰도↑ |
| 유동성 | 일평균 거래대금 100억+ | 슬리피지 최소화 |
| RSI(2) | 현재 RSI(2) ≤ 5 (공격적) 또는 ≤ 10 (보수적) | 스크리닝 핵심 |

#### 매수 타이밍
```
[기본 전략]
- 종가 > 200일 SMA
- RSI(2) ≤ 5 → 당일 종가에 매수 (MOC 주문)
- 종목 수가 많으면 RSI(2)가 가장 낮은 순서로 우선

[Connors RSI 변형]
- CRSI(3,2,100) ≤ 15 → 매수
  * CRSI = [RSI(3) + RSI(UpDown_Streak, 2) + PercentRank(ROC(1), 100)] / 3

[누적 RSI 변형]
- 2일간 RSI(2) 합계 < 10 → 매수 (더 극단적 과매도)
```

#### 매도 타이밍
```
[기본 청산]
- 종가 > 5일 SMA → 익일 시가에 매도

[Connors RSI 청산]
- CRSI(3,2,100) ≥ 85 → 매도

[시간정지]
- 진입 후 5거래일 경과 시 무조건 청산

[손절]
- 진입가 대비 -5% (대형주 기준)
- 또는 ATR(14) × 3.0
```

### 5. 풀백(눌림목 매수) 전략

#### 종목 선정
| 조건 | 기준 | 파이썬 조건 |
|------|------|-----------|
| 장기 추세 | 종가 > 200일 SMA | `close > sma_200` |
| 중기 추세 | 50일 SMA > 200일 SMA | `sma_50 > sma_200` |
| 상대강도 | 3개월 수익률 상위 20% | `pct_rank(roc_63) > 0.8` |
| 풀백 감지 | 종가 < 20일 SMA | `close < sma_20` |
| RSI 조건 | 5일 RSI < 45 | `rsi_5 < 45` |
| **한국 추가** | 외국인+기관 5일 순매수 양전환 | 수급 반전 신호 |

#### 매수 타이밍
```
[기본 풀백 매수]
1. 종가 > 200일 SMA (추세 확인)
2. 종가 < 20일 SMA (풀백 진행 중)
3. RSI(5) < 45 (과매도 접근)
4. 당일 양봉 출현 (반등 시작 확인) → 익일 시가 매수

[Holy Grail 변형 (Linda Raschke)]
1. ADX(14) > 30 (강한 추세 확인)
2. 가격이 20일 EMA까지 풀백
3. 트리거 바(=풀백 저점 이후 첫 양봉) 고가 위에 매수 스톱 주문
   → 체결되면 진입, 미체결 시 익일 업데이트

[Minervini VCP 풀백]
1. 추세 템플릿 8가지 조건 모두 충족 (종목선정 단계)
2. 변동성 수축 패턴: 직전 3~4회 조정의 깊이가 점차 감소
   (예: -20% → -12% → -6% → -3%)
3. 마지막 수축 구간의 고가 돌파 + 거래량 > 50일 평균 × 1.5
```

#### 매도 타이밍
```
[익절]
- RSI(5) > 65 → 매도 (기본)
- 또는 종가 > 10일 SMA 회복 시 (보수적)
- 또는 수익률 +8~12% 도달 시 50% 청산, 나머지 트레일링

[트레일링 스톱]
- 10일 SMA 또는 20일 EMA 하향 이탈 시 전량 청산
- ATR(14) × 2.0~3.0 트레일링

[손절]
- 최근 스윙 저점 하방 1~2% 또는 ATR × 2.0
- 고정 손절: -7~8% (Minervini 기준)
- 진입 후 5~10거래일 수익 없으면 시간정지 청산
```

### 6. 브레이크아웃 전략

#### 종목 선정
| 조건 | 기준 |
|------|------|
| 박스권 형성 | 20~60일간 가격 변동 범위 < ATR×10 (좁은 레인지) |
| 거래량 감소 | 박스권 내 거래량이 점진적 감소 (에너지 축적) |
| 추세 방향 | 박스권 이전 상승추세 (계속 패턴 > 반전 패턴) |
| 52주 고가 근접 | 현재가가 52주 고가의 95% 이내 |
| **한국** | 테마·섹터 리더 (동종업계 최초 돌파 시도 종목) |

#### 매수 타이밍
```
[박스권 돌파]
1. 20~60일 최고가를 종가 기준으로 돌파
2. 돌파일 거래량 > 20일 평균 × 1.5 (필수 필터)
3. 돌파 캔들이 박스 상단 위에서 마감 (위꼬리 최소)
→ 돌파 확인 후 익일 시가 매수

[52주 신고가 돌파]
1. 종가 > 52주 최고가 × 1.015 (1.5% 이상 돌파)
2. 거래량 > 20일 평균 × 1.5
3. RSI(14) > 50 (모멘텀 동반)
→ 돌파 확인 후 매수

[풀백 진입 (더 안전)]
1. 초기 돌파 확인
2. 3~5일 내 이전 저항선(=현 지지선)까지 풀백
3. 지지선에서 양봉 반등 + 거래량 감소 확인
→ 양봉 확인 후 매수
```

#### 매도 타이밍
```
[익절]
- 1차 목표: 박스 높이만큼의 측정이동 (Measured Move)
  예: 박스 상단 50,000원, 하단 45,000원 → 목표 55,000원
- 2차 목표: 피보나치 확장 161.8%
- ATR × 3.0~4.0 트레일링 스톱 병행

[손절]
- 돌파 기준선 (박스 상단) 하회 시 즉시 청산
- 또는 ATR × 1.5 하방
- 거짓 돌파 판별: 돌파 후 2일 내 박스 안으로 복귀 시 손절

[시간정지]
- 돌파 후 5거래일 내 추가 상승 없으면 청산 검토
```

### 7. 이동평균선 크로스오버 (골든크로스/데드크로스)

#### 종목 선정
| 조건 | 기준 |
|------|------|
| 대상 | 지수 ETF (SPY, QQQ, KODEX200, TIGER KOSDAQ150) |
| 또는 | 시가총액 상위 대형주 (삼성전자, AAPL 등) |
| 이유 | MA 크로스는 개별 중소형주에서 노이즈가 심함 |
| 거래 빈도 | 연 2~4회 (매우 낮음) → 세금 부담 최소 |

#### 매수 타이밍
```
[기본: 50/200 SMA 골든크로스]
- 50일 SMA가 200일 SMA를 상향 돌파
- 확인: 3거래일 연속 50 SMA > 200 SMA 유지 시 진입
  (1일 돌파만으로는 거짓 신호 많음)

[개선: 가격+MA 복합 필터]
- 50 SMA > 200 SMA (골든크로스)
- 종가 > 50 SMA (가격도 MA 위에 위치)
- 200 SMA 기울기 > 0 (장기 추세 상승 확인)
→ 세 조건 모두 충족 시 매수

[한국 시장 변형]
- KOSPI: 20일/60일 MA 크로스가 더 민감하게 작동
- 골든크로스 + 외국인 순매수 전환 동시 발생 시 신뢰도↑↑
```

#### 매도 타이밍
```
[기본 청산]
- 50일 SMA가 200일 SMA를 하향 돌파 (데드크로스) 시 전량 매도
- 또는 종가가 200일 SMA 하회 시 (더 빠른 청산)

[개선 청산]
- 종가가 50일 SMA 하회 + RSI(14) < 40 → 청산
- 횡보장 회피: 50 SMA와 200 SMA 간격이 0.5% 미만이면
  크로스 신호 무시 (휩소 방지)

[손절]
- 없음 (크로스 반전이 곧 손절)
- 또는 보수적으로 200일 SMA 하회 시 즉시 청산
```

### 8. RSI/스토캐스틱 과매도 반등 (스윙)

#### 종목 선정
| 조건 | 기준 |
|------|------|
| 장기 추세 | 종가 > 200일 SMA (상승추세에서만) |
| RSI(14) | 30 이하 (과매도 진입) |
| 기본적 분석 | PER < 업종 평균, 부채비율 < 100% (가치 함정 회피) |
| 거래량 | 과매도 구간에서 거래량 급증 (셀링 클라이맥스) |
| **한국** | 외국인 순매도 5일+ 연속 후 순매수 전환일 |

#### 매수 타이밍
```
[RSI(14) 과매도 반등]
1. RSI(14) ≤ 30 도달 확인
2. RSI가 30 아래에서 다시 30 위로 상향 돌파 시 매수
   (30 아래에서 바로 매수 X → 반등 확인 후 매수)
```

### ATR 기반 포지션 사이징 공식
```python
def calc_position_size(account, risk_pct, entry, stop, atr=None):
    """
    account: 계좌 자산
    risk_pct: 거래당 리스크 비율 (0.01 = 1%)
    entry: 진입 예정가
    stop: 손절가 (없으면 ATR 기반 계산)
    """
    dollar_risk = account * risk_pct
    if stop:
        stop_distance = abs(entry - stop)
    else:
        stop_distance = atr * 2.0  # ATR × 2배를 손절 거리로
    
    shares = int(dollar_risk / stop_distance)
    position_value = shares * entry
    
    # 계좌의 25% 초과 단일 포지션 방지
    max_shares = int(account * 0.25 / entry)
    return min(shares, max_shares)
```
